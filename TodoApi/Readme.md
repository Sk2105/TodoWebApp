# Todo api for todo app

## Discription

Todo api for todo app using spring boot and maven 

## How to use

1. run `mvnw spring-boot:run`
2. open `http://localhost:8080`


## Endpoints

- `/todo`: get all todos
- `/todo`: post new todo
- `/todo/{id}`: delete todo by id
- `/todo/{id}`: update todo by id
